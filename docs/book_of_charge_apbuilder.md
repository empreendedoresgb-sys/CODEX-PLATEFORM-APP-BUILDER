## Product principle
One sentence should generate one complete, high-quality functional app with minimal manual intervention.

1. Requirement parsing

5. Deployment preview

- Multi-agent orchestration
- Automated UI/backend/data generation
- Security and quality validation gates
- Preview URL generation
- Registry-driven language support
- API-first architecture

## Manual intervention policy
Manual actions only for unavoidable governance operations:
- production deployment approval
- legal acceptance
- billing activation
